# GEAR UP AI-Powered Student Website Project

Collaborative student web development project built with HTML, CSS, JavaScript, GitHub, and AI-assisted coding.

## Project Overview

This repository includes one central landing page and five linked personal websites:

- 4 student websites
- 1 project lead website

The central hub explains the project mission, learning goals, and AI workflow.

## Learning Goals

- Build foundational skills in HTML, CSS, and JavaScript
- Practice collaboration with GitHub version control
- Use AI tools responsibly to assist coding and debugging
- Create creative, interactive portfolio websites

## AI Integration Guidelines

Students can use AI tools to:

- Generate starter code
- Suggest design/layout improvements
- Debug code issues
- Improve writing content

Students should always:

- Understand the code they use
- Be able to explain how their code works
- Personalize AI-generated content and style

## Run Locally

1. Open `index.html` in a web browser.
2. Use VS Code Live Server (optional) for auto-reload while editing.

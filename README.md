## Website Starter

This project now includes a normal responsive website starter built with plain HTML, CSS, and JavaScript.

### Files

- `index.html` - page structure and sections
- `styles.css` - responsive styling
- `script.js` - mobile menu toggle and contact form message

### Run locally

1. Open `index.html` directly in your browser.
2. (Optional) Use VS Code Live Server for auto-reload while editing.

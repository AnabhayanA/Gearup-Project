About the Project

This project helps students build early digital portfolios while practicing coding, collaboration, and design. We use GitHub for version control and teamwork, and AI tools to support learning, debugging, and creative layout planning.

  • HTML structure and semantic sections
  • CSS styling, responsive layouts, and visual design
  • JavaScript interactivity and UI behavior
  • GitHub workflow and collaborative development

How We Used AI

GitHub Copilot helped our team suggest starter code, debug errors, improve page structure, and generate fresh layout ideas. Students reviewed, edited, and personalized each AI-generated part to make sure they understood how the code works.

Code Suggestions

Generated starter blocks for HTML, CSS, and JavaScript.

Debugging Support

Helped identify syntax mistakes and logic issues faster.

Layout Ideas

Suggested clean section structure and responsive design patterns.

Content Improvement

Improved wording, clarity, and tone for student portfolio sections.
